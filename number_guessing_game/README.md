# 🎮 Number Guessing Game (Python Learning Project)

This project is a number guessing game built in Python.  
It demonstrates my learning progress from basic Python concepts to more advanced program structure and input validation.

---

# 🚀 Project Overview

The player must guess a randomly generated number within a chosen difficulty level.

The game includes:
- Difficulty selection
- Limited attempts
- Score system
- Input validation
- Replay option
