# Password Checker 🔐

A simple Python program that checks whether a password is strong or weak based on basic rules.

---

## 🎯 What it does

The program asks the user to enter a password and checks it using the following rules:

- At least 8 characters long
- Contains at least one number
- Contains at least one uppercase letter

---

## ▶️ How to run

```bash
python password_checker.py
```

---

## 🧪 Example

```
Enter password: hello
Too short

Enter password: hello123
Password needs uppercase

Enter password: Hello123
Password OK
```

## 🧠 Skills used

- Variables
- User input
- Conditional statements (`if / elif / else`)
- String methods (`isdigit()`, `isupper()`)
- Length checking (`len()`)
- Logical flow control
